# 🔐 KeyKeep-Bot

**KeyKeep-Bot** GeoMind is a lightweight bot that uses an API to find out the population, country, city, weather (NOT YET), and the country you enter.

## 🛠 Commands
- `/start` - View the welcome message and basic instructions.
- `/info [country]` - find out information about the country

> [!IMPORTANT]
> **about the weather:**
> Weather has not been added yet. It will be added in the next update.

## ⚙️ Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Luto228/KeyKeep-Bot.git
   ```
2. **Install dependencies:**
   ```bash
   pip install aiogram
   ```
3. **Configure the bot:**
   - Locate the file `config.py.example`.
   - Rename it to `Config.py`.
   - Open `Config.py` and replace `"YOUR_TOKEN_BOT"` with your actual Telegram Bot Token.
4. **Run the bot:**
   ```bash
   python bot.py
   ```
